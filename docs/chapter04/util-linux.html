4.13. Util-Linux-2.42

The Util-linux package contains miscellaneous utility programs.

Package Information

  • Download HTTP: https://www.kernel.org/pub/linux/utils/util-linux/v2.42/util-linux-2.42.tar.xz

  • Download SHA256 Sum: 3452b260bbaa775d6e749ac3bb22111785003fc1f444970025c8da26dfa758e9

  • Download Size: 10,360 KB

  • Estimated Disk Space: 149.3M

  • Estimated Build Time: less than 0.5 SBU

4.13.1. Installation of Util-linux

Prepare Util-linux for compilation:

Now Configure the Package, issue: 

mkdir -pv /var/lib/hwclock
./configure --libdir=/usr/lib --runstatedir=/run \
            --disable-chfn-chsh --disable-login \
            --disable-nologin --disable-su \
            --disable-setpriv --disable-runuser \
            --disable-pylibmount --disable-static \
            --disable-liblastlog2 --without-python \
            ADJTIME_PATH=/var/lib/hwclock/adjtime \
            --docdir=/usr/share/doc/util-linux-2.42

To Compile the Package, issue: 

make

Now Install the Package, issue: 

make install
