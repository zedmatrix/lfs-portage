4.23. Libcap-2.78

The Libcap package implements the userspace interface to the POSIX 1003.1e capabilities available in Linux kernels.

Package Information

  • Download HTTP: https://www.kernel.org/pub/linux/libs/security/linux-privs/libcap2/libcap-2.78.tar.xz

  • Download SHA256 Sum: 0d621e562fd932ccf67b9660fb018e468a683d7b827541df27813228c996bb11

  • Download Size: 196 KB

  • Estimated Disk Space: 1.2M

  • Estimated Build Time: less than 0.5 SBU

4.23.1. Installation of Libcap

Prepare Libcap for compilation:

Now Configure the Package, issue: 

sed -i '/install -m.*STA/d' libcap/Makefile

To Compile the Package, issue: 

make prefix=/usr lib=lib

To Test the Package, issue: 

make test

Now Install the Package, issue: 

make prefix=/usr lib=lib install
