4.11. Texinfo-7.3

The Texinfo package contains programs for reading, writing, and converting info pages.

Package Information

  • Download HTTP: https://mirror.csclub.uwaterloo.ca/gnu/texinfo/texinfo-7.3.tar.xz

  • Download SHA256 Sum: 51f74eb0f51cfa9873b85264dfdd5d46e8957ec95b88f0fb762f63d9e164c72e

  • Download Size: 6,778 KB

  • Estimated Disk Space: 106.3M

  • Estimated Build Time: less than 0.5 SBU

4.11.1. Installation of Texinfo

Prepare Texinfo for compilation:

Now Configure the Package, issue: 

./configure --prefix=/usr

To Compile the Package, issue: 

make

Now Install the Package, issue: 

make install
