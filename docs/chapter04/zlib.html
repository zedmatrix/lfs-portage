4.4. Zlib-1.3.1

The Zlib package contains compression and decompression routines used by some programs.

Package Information

  • Download HTTP: https://zlib.net/fossils/zlib-1.3.1.tar.gz

  • Download SHA256 Sum: 9a93b2b7dfdac77ceba5a558a580e74667dd6fede4585b91eefb60f03b72df23

  • Download Size: 1,477 KB

  • Estimated Disk Space: 5.4M

  • Estimated Build Time: less than 0.5 SBU

4.4.1. Installation of Zlib

Prepare Zlib for compilation:

Now Configure the Package, issue: 

./configure --prefix=/usr

To Compile the Package, issue: 

make

To Test the Package, issue: 

make check

Now Install the Package, issue: 

make install
rm -fv /usr/lib/libz.a
