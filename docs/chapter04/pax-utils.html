Pax-utils Dependencies

Required

meson-1.11.1 ninja-1.13.2

4.27.1. Installation of Pax-utils

Prepare Pax-utils for compilation:

Now Configure the Package, issue: 

meson setup build --prefix=/usr -Dlddtree_implementation=sh \
  -Dtests=false -Duse_fuzzing=false

To Compile the Package, issue: 

ninja -C build

Now Install the Package, issue: 

ninja -C build install
