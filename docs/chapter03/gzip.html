3.10. Gzip-1.14

The Gzip package contains programs for compressing and decompressing files.

Package Information

  • Download HTTP: https://mirror.csclub.uwaterloo.ca/gnu/gzip/gzip-1.14.tar.xz

  • Download SHA256 Sum: 01a7b881bd220bfdf615f97b8718f80bdfd3f6add385b993dcf6efd14e8c0ac6

  • Download Size: 865 KB

  • Estimated Disk Space: 9.9M

  • Estimated Build Time: less than 0.5 SBU

Gzip Dependencies

Required

Grep-3.12

3.10.1. Installation of Gzip

Prepare Gzip for compilation:

Now Configure the Package, issue: 

./configure --prefix=/usr --host=$LFS_TGT

To Compile the Package, issue: 

make

Now Install the Package, issue: 

make DESTDIR=$LFS install
