2.3. Linux-headers-7.0

The Linux API Headers (in linux-7.0.tar.xz) expose the kernels API for use by libc

Package Information

  • Download HTTP: https://cdn.kernel.org/pub/linux/kernel/v7.x/linux-7.0.tar.xz

  • Download SHA256 Sum: bb7f6d80b387c757b7d14bb93028fcb90f793c5c0d367736ee815a100b3891f0

  • Download Size: 153,461 KB

  • Estimated Disk Space: 1.5G

  • Estimated Build Time: less than 0.5 SBU

Linux-headers Dependencies

Required

Binutils-2.46.0-pass1 GCC-15.2.0-pass1

2.3.1. Installation of Linux-headers

Prepare Linux-headers for compilation:

Now Configure the Package, issue: 

make mrproper

To Compile the Package, issue: 

make headers

Now Install the Package, issue: 

find usr/include -type f ! -name '*.h' -delete

cp -rv usr/include $LFS/usr
